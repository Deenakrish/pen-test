# Glossary

**ARP** – Address Resolution Protocol used for resolving IP-to-MAC mappings.  
**OSINT** – Open-Source Intelligence gathering from publicly accessible sources.  
**Enumeration** – Systematic probing to discover services, users, and shares.  
**CVSS** – Common Vulnerability Scoring System used for rating severity.  
**Exploit** – A technique or code that leverages a vulnerability for system compromise.  
**PoC** – Proof of Concept demonstrating exploitability.
